Medical Education RAG Chatbot (Flagship)

An interactive medical learning chatbot powered by Retrieval-Augmented Generation (RAG), vector databases, and LLaMA 2. Designed to help students quiz themselves using textbook content in a conversational format.

Core Components

  • Retrieval Augmented Generation (RAG): Quizzes students based on textbook embeddings, improving retention through generative flashcards.
  • Vector Database: Efficient retrieval of relevant content using ChromaDB and document chunking.
  • LLaMA 2 Integration: Generates detailed, domain-specific answers aligned with medical learning objectives.
  • Interactive UX: Enables self-paced, conversational learning with contextual memory and feedback loops.

Tools and Technologies

LLaMA 2, Chroma DB, FAISS, HuggingFace Transformers, Streamlit (optional), Python, LangChain

Educational Impact

  • Promotes active recall and spaced repetition using conversational quizzes.
  • Improves understanding by linking textbook sections with interactive prompts.
  • Supports lifelong learning with personalized flashcard experiences.
